# Centro Renal SPA · Guía Rápida

## Glosario de campos
- **Rut**: Identificador único del funcionario.
- **Tipo de registro**: Licencia Médica, Permiso, Vacaciones, Descanso Compensatorio o Turno.
- **Subtipo**: Detalle específico (ej.: Reposo, Permiso sin goce, Turno Noche).
- **Estado**: Ciclo de aprobación (Aprobado, Pendiente, Rechazado).
- **Días / Horas**: Se calculan automáticamente según reglas configuradas.
- **Sede**: Quilpué, Villa Alemana o Viña del Mar.

## Cómo usar el dashboard
1. Sube o selecciona el archivo base desde la barra lateral.
2. Ajusta el mapeo de columnas si tus encabezados difieren del estándar.
3. Aplica filtros por sede, persona, fechas y tipo de registro para analizar segmentos específicos.
4. Revisa las tarjetas KPI para conocer el estado general.
5. Navega entre pestañas: **Resumen** (gráficos), **Tabla** (detalle y descargas), **Turnos** (análisis específico) y **Configuración** (umbrales y reglas).
6. Genera exportaciones en PDF o Excel con los filtros actuales.

## Tips
- Usa el botón **Limpiar filtros** para volver al estado inicial.
- Cambia a **Modo oscuro** desde Configuración para salas de control con poca luz.
- Los umbrales de semáforo pueden ajustarse para alinearse con la política interna.
- El heatmap de turnos ayuda a detectar sobrecarga en personas o meses específicos.
